## 📖 Overview

Travel Website is a modern, interactive, and user-friendly web application designed for exploring destinations, discovering tour packages, and planning unforgettable trips.

This project focuses on clean UI/UX, responsive design, smooth animations, and scalable React architecture.

# ⚙️ Installation

Clone repository

```bash
git clone https://github.com/gopanmodak/WanderNest
```

Move to project folder

```bash
cd travel-website
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
